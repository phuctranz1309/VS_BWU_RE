# 🎸 VS BoyWithUke - REBOOTED (BETA)

Welcome to the GitHub repository for **VS BoyWithUke - REBOOTED**. This is a passion project bringing the music of BoyWithUke into the world of *Friday Night Funkin'*.

> **Note:** This repository is used for hosting the mod files. It does not contain the source code of the engine.

---

## ✨ About the Project
This "Rebooted" version is a humble attempt to improve the original mod. I've focused on cleaning up the visuals and refining the gameplay to give players a better experience. 

It is still in **BETA**, so some features are a work in progress!

### 🛠 What’s New in 1.2.0 (BETA)
* **HD Sprites:** New "Unpixelated" BWU sprites (Big thanks to *Dessher* and *doccys*).
* **Remastered Tracks:** "Toxic", "IDGAF", "Senpai (BWU take)", and "Titular" have been re-charted.
* **Engine Upgrade:** Optimized for **P-Slice Engine** (Recommended) and **Psych Engine 1.0+**.

## 📥 Installation
To play this mod, you **NEED** to know how to import mods into **P-Slice Engine** or **Psych Engine**.

1.  Download the `.zip` file from the [Releases](https://github.com/phuctranz1309/VS_BWU_RE/releases) section.
2.  Extract the folder.
3.  Move the extracted folder into your P-Slice engine's `mods` directory.
4.  Launch the game and enjoy!
